Frequently Asked Questions

How to use the Date Picker?
The date picker allows the user to select a date and time from a popup calendar. When opeing up the data picker firts pick the specific date you are interested in then pick the time. The time picker can be viewed by clicking on the small circular icon at the bottom center of the date picker.

How to Edit a profile photo?
When editing a profile you can select a custom photo from a file. Once the profile has been saved you can then view your updated profile which will include your photo. If you go back and edit the profile again you will notice that it looks as if you no longer have your profile photo. This is not the case. Your original profile photo will still be there so long as you don't make a changes to the profile photo during the edit process. Your photo hasn't gone anywhere and it is still viewable in your profile.

Who can view the Organization page?
Everything on the organization profile is publicly viewable except for the "Edit" function. This is made only available to the delegated organization adminstrators
